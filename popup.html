Welcome to bookmarks backup!!

This is an browser extension, which will be used to backup bookmarks and restore bookmarks from one different browser to another browser without any login. This Extension has two options Download, and Restore. When User Clicks Download a csv file will be downloaded which contains all the bookmarks title and urls, will work on any broswer. User can upload this file in any other browser to get the favorites in other browser using this same extension restore functionlaity.
